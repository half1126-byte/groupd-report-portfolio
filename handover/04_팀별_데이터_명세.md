# 팀별 데이터 입출력 명세

## 1. 예약팀 (reservation)

### 입력 파일
- **파일명 패턴**: `*예약자관리*.xlsx`, `*예약*관리*.xlsx`
- **프로세서**: `src/processors/reservation.py`

### 수집 컬럼

| 엑셀 컬럼명 | 용도 |
|-------------|------|
| `예약번호` | 예약 건수 집계 |
| `유입경로` | 유입 경로 정규화 → TOP5 |
| `상태` | 완료/취소 분류 |
| `예약신청일시` | 월별 분류 |
| `예약취소일시` | 취소 건수 |
| `취소사유` | 취소 사유 TOP5 |
| 진료 컬럼 (자동 감지) | 스케일링, 임플란트, 교정 등 → 희망 진료 TOP5 |

### 출력 KPI

```python
'kpi': {
    'actual_reservations': int,       # 내원 확정 수
    'prev_actual_reservations': int,  # 전월 내원 확정
    'total_reservations': int,        # 총 예약 신청
    'canceled_count': int,            # 취소/노쇼
}
```

### 보고서 출력
- 메트릭: 총 신청, 내원 확정, 취소/노쇼 (전월 vs 당월)
- TOP5 × 3: 희망 진료, 유입경로, 취소 사유

## 2. 블로그팀 (blog)

### 입력 파일 (3종)

| 파일명 패턴 | 내용 |
|-------------|------|
| `[콘텐츠팀] 포스팅 업무 현황*.csv` | 계약/발행/이월 건수, 포스팅 제목/URL |
| `유입분석_월간_*.xlsx` | 유입 키워드, 트래픽 소스 |
| `조회수_월간_*.xlsx` / `조회수_순위_월간_*.xlsx` | 게시물별 조회수 |

### 프로세서: `src/processors/blog.py` (1,130줄 — 가장 복잡)

### 주요 처리 로직

- **15일 기준 월 분류**: 계약 시작일이 15일 이전이면 해당월, 이후면 다음월
- **이월 계산**: 전월 미완료분 → 당월 이월
- **조회수 집계**: 게시물별 월간 조회수 합산
- **유입 키워드 파싱**: URL 쿼리 파라미터에서 검색어 추출

### 출력 KPI

```python
'kpi': {
    'contract_count': int,         # 계약 건수
    'published_count': int,        # 발행 완료
    'carryover_count': int,        # 이월 건수 ★ 실행계획 예산에 사용
    'total_views': int,            # 총 조회수
    'views_mom_growth': float,     # 전월 대비 조회수 증감률(%)
    'publish_completion_rate': float,  # 완료율
    'prev_published_count': int,
    'prev_contract_count': int,
    'prev_carryover_count': int,
    'prev_total_views': int,
}
```

### 보고서 출력
- 메트릭 2행: (조회수, 계약) + (발행, 이월)
- TOP5 × 3: 조회수 TOP5, 유입 키워드 TOP5, 유입경로 TOP5
- Key Insights: 최고 조회 게시물, 1위 키워드
- 포스팅 리스트: 제목 + URL + 날짜 (전월/당월)

## 3. 광고팀 (ads)

### 입력 파일 (2종)

| 파일명 패턴 | 내용 |
|-------------|------|
| `날짜_치과명_광고비 소진 내역.xlsx` | 월별 광고비 소진액 |
| `날짜_치과명_기간별 내역.csv` | 캠페인별 키워드, 노출수, 클릭수 |

### 프로세서: `src/processors/ads.py`

### ⚠️ 의존성: 예약팀 데이터 필요

```python
# CPA = 광고비 / 내원 확정 수
ads_result = process_ads(files['ads'], reservation_result)
```

### 출력 KPI

```python
'kpi': {
    'total_impressions': int,          # 총 노출수
    'impressions_mom_growth': float,   # 노출수 증감률
    'total_clicks': int,               # 총 클릭수
    'cpa': float,                      # CPA (광고비/내원수)
    'prev_total_impressions': int,
    'prev_cpa': float,
}
```

### 보고서 출력
- 메트릭: 노출수, 클릭수 (전월 vs 당월)
- CPA 배너: 건당 광고 비용 + 전월 대비 증감
- 키워드 4분면: 전월 노출/클릭 TOP5, 당월 노출/클릭 TOP5

## 4. 디자인팀 (design)

### 입력 파일
- **파일명 패턴**: `[디자인팀] 업무 협조 요청*.csv/xlsx`
- **프로세서**: `src/processors/design.py`

### 수집 컬럼

| 엑셀 컬럼명 | 용도 |
|-------------|------|
| `의뢰일` / `업무기한` | 월별 분류 |
| `완료 페이지` | 총 페이지 수 |
| `수정 횟수` | 평균 수정 횟수 |
| `상태` | "완료" 포함 → 완료 건수 |
| `치과명` | 멀티 치과 필터링 |
| `디자인 작업명` / `분류` | 작업 유형별 집계 |

### 출력 KPI

```python
'kpi': {
    'total_tasks': int,       # 총 작업 수
    'completed_tasks': int,   # 완료 수
    'total_pages': int,       # 총 페이지
    'avg_revisions': float,   # 평균 수정 횟수
}
```

### 보고서 출력
- 작업 요약 테이블: 유형별 전월/당월 건수 + 평균 수정 횟수
- 상세 리스트: 작업명, 수정 횟수, 페이지 수

### 특이사항
- `clinic_name` 필터: 해당 치과 데이터 없으면 보고서에서 섹션 자체 제외

## 5. 영상팀 (youtube)

### 입력 파일 (3종)

| 파일명 패턴 | 내용 |
|-------------|------|
| `[영상팀] 업무 현황*.csv` | 계약, 완료, 상태, 기획→촬영→편집→업로드 날짜 |
| `*유튜브 콘텐츠 DB.xlsx` | 영상 메타데이터 |
| `*유튜브 트래픽 DB.xlsx` | 트래픽 소스 |

### 프로세서: `src/processors/youtube.py`

### 출력 KPI

```python
'kpi': {
    'total_contracts': int,     # 계약 건수
    'completed_count': int,     # 제작 완료
    'completion_rate': float,   # 완료율
    'views': int,               # 총 조회수
    'views_mom_growth': float,  # 조회수 증감률
    'subscribers': int,         # 구독자 수
}
```

### 보고서 출력
- 메트릭: 조회수, 시청 시간, 구독자 증감
- 제작 현황: 계약/완료 건수
- TOP3: 조회수 TOP3 동영상, 트래픽 소스 TOP3
- 영상 유형 분포: 롱폼 vs 숏폼

## 6. 기술설정팀 (setting)

### 입력 파일
- **파일명 패턴**: `세팅KPI*.csv`, `신규계약프로세스*.csv`
- **프로세서**: `src/processors/setting.py`

### 출력 구조

```python
{
    'avg_progress_rate': 75.5,   # 전체 평균 진행률
    'completed_clinics': 5,      # 완료 치과 수
    'risk_clinics': 2,           # 미비 치과 수
    'total_clinics': 8,
    'clinic_progress': [{
        'clinic': '서울리멤버치과',
        'progress_rate': 80.0,
        'platform_groups': [{
            'platform': '네이버',
            'completion_rate': 85.0,
            'channels': [{
                'channel': '네이버 블로그',
                'status': 'completed',
                'sub_tasks': [...]
            }]
        }]
    }]
}
```

### 보고서 출력
- 전체 진행률 + 완료/미비 치과 수
- 치과별 프로그레스 바
- 플랫폼별(네이버, 구글 등) 채널 완료 상태
