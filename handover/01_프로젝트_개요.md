## 2. 이 앱은 무엇인가?

치과 마케팅 대행사(그룹디)의 PM이 **월간 마케팅 보고서**를 자동 생성하는 도구입니다.

### 핵심 기능
1. **엑셀/CSV 업로드** → 6개 팀 데이터를 자동 분류
2. **KPI 계산** → 전월 대비 성장률, TOP5 랭킹 자동 생성
3. **HTML 보고서 생성** → 고객(치과 원장님)에게 전달할 보고서
4. **실행 계획(액션플랜)** → PM이 카탈로그에서 상품을 선택하면 보고서에 포함
5. **AI 요약** → 팀장용 3줄 요약 자동 생성 (Claude/GPT)

### 사용 흐름
```
PM이 각 팀 엑셀 업로드
    → 데이터 자동 분류 & 처리
    → [보고서 미리보기] 탭에서 확인
    → [데이터 확인 및 수정] 탭에서 수정
    → [실행 계획 편집] 탭에서 액션플랜 선택
    → AI 요약 생성 (선택)
    → HTML 보고서 다운로드
```

## 3. 6개 팀 데이터 요약

| 팀 | 입력 파일 | 보고서에 표시되는 내용 |
|----|-----------|----------------------|
| **예약팀** | `*예약*.xlsx` | 예약 건수, 내원 확정, 취소/노쇼, 희망 진료 TOP5, 유입경로 TOP5 |
| **블로그(콘텐츠)팀** | `[콘텐츠팀] 포스팅*.csv` + `유입분석_월간_*.xlsx` + `조회수_*.xlsx` | 조회수, 계약/발행/이월 건수, 키워드 TOP5, 포스팅 리스트 |
| **광고팀** | `날짜_치과명_광고비 소진*.xlsx` + `날짜_치과명_기간별*.csv` | 노출수, 클릭수, CPA, 키워드 랭킹 4분면 |
| **디자인팀** | `[디자인팀] 업무 협조 요청*.csv/xlsx` | 작업 유형별 건수, 수정 횟수, 페이지 수 |
| **영상(유튜브)팀** | `[영상팀] 업무 현황*.csv` + `유튜브 콘텐츠/트래픽 DB.xlsx` | 조회수, 시청 시간, 구독자, 제작 현황, 트래픽 소스 |
| **기술설정팀** | `세팅KPI*.csv` | 플랫폼별 세팅 진행률, 치과별 완료 현황 |

### 팀 간 데이터 의존성
- **광고팀 → 예약팀**: CPA 계산에 예약팀 "내원 확정 수" 필요
- 나머지는 모두 독립

## 4. 주요 용어

| 용어 | 설명 |
|------|------|
| **이월(carryover)** | 전월에 완료하지 못한 블로그 포스팅 건수가 당월로 넘어온 것 |
| **치환건** | 이월 1건 = 0.5 치환건. 대체상품으로 교환할 수 있는 단위 |
| **건당 대체** | 블로그 1건(20만원) 대신 다른 마케팅 상품으로 교체 시 해당 상품의 수량 |
| **PM 제안** | PM이 직접 제안하는 추가 마케팅 상품 |
| **카탈로그 제안** | 엑셀 카탈로그에서 불러온 대체상품 목록 |
| **desc** | 각 상품의 설명문 (보고서에 표시됨) |
| **계약포함** | 기존 계약 범위 내 상품 / **추가제안** | 별도 비용 발생 상품 |
